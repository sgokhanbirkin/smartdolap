# Sound Assets

## Dit Sound (dit.mp3)

This folder should contain a `dit.mp3` file for barcode scan feedback.

### How to Add the Sound File

#### Option 1: Download from Free Sound Libraries
- **Freesound.org**: <https://freesound.org/search/?q=beep+short>
- **Zapsplat**: <https://www.zapsplat.com/sound-effect-category/beeps-and-blips/>
- Search for: "short beep", "scanner beep", "dit sound", "cash register beep"

#### Option 2: Generate Your Own
Use online tools like:
- **Bfxr**: <https://www.bfxr.net/> (Game sound generator)
- **Audacity**: Free audio editor to create custom sounds

#### Option 3: Use System Sound (Current Fallback)
If `dit.mp3` is not found, the app will automatically fall back to system click sound.

### Recommended Sound Characteristics
- **Duration**: 50-150ms (very short)
- **Format**: MP3 (for compatibility)
- **Sample Rate**: 44.1kHz
- **Bit Rate**: 128kbps
- **Volume**: Medium (not too loud)
- **Tone**: High-pitched "beep" or "dit" sound

### Example Sound Description
Think of:
- Barcode scanner "beep" at supermarket checkout
- Credit card terminal confirmation sound
- Short, pleasant, non-intrusive tone

### File Location
Place the file at: `assets/sounds/dit.mp3`

The app will automatically detect and use it!
